11 — Conteúdo

Listas de Features

Padrão extraído das seções de dores e módulos do site.

Falta de mentalidade empreendedora — Ainda pensa como prestador de serviço, não como dono de negócio.
Gestão financeira frágil — Não sabe quanto cobrar, mistura finanças e trabalha sem lucro real.
Equipe desorganizada — Não sabe contratar, treinar ou liderar e acaba sobrecarregado.
Marketing ineficiente — Depende de indicação, não sabe se posicionar e não atrai o cliente certo.
Salão sem identidade — Não tem diferencial, não fideliza, vive na incerteza.
Falta de processo e estrutura — Tudo funciona no improviso, sem planejamento e com desgaste constante.

17 — Referência

Tokens CSS Completos

Copie o bloco abaixo e cole no :root do seu projeto.

/* ══════════════════════════════════════════════════════
DESIGN SYSTEM — MÉTODO ÁGUIA · ALESSANDRA SALUCCI
v1.1 · 2025 — Fonte: post-5.css (kit) + post-13.css (página)
══════════════════════════════════════════════════════ */

/* Fontes: Adobe TypeKit pft6rxt */
@import url("https://use.typekit.net/pft6rxt.css");

:root {
  /* ── Cores · post-5.css (Elementor Kit 5) ── */
  --color-primary:        #9D2D19; /* --e-global-color-primary */
  --color-primary-dark:   #5C1F14; /* --e-global-color-aa68f8d */
  --color-salmon:         #ED7F6C; /* --e-global-color-6f59f7e */
  --color-peach:          #E2AA80; /* --e-global-color-5291032 */
  --color-gold:           #FFBC7D; /* page-transition color */
  --color-taupe:          #997E69; /* --e-global-color-6c043ad */
  --color-accent:         #25241E; /* --e-global-color-accent · BG única seção escura (#entrar) */
  --color-gray-700:       #54595F; /* --e-global-color-secondary */
  --color-gray-500:       #7A7A7A; /* --e-global-color-text */
  --color-gray-200:       #E9E9E9; /* --e-global-color-51a296d */
  --color-gray-100:       #DDDDDD; /* --e-global-color-2efd0a5 */
  --color-white:          #FFFFFF; /* --e-global-color-a8511bc */

  /* ── Gradientes · post-13.css (valores literais da página) ── */
  /* gradient-text1: taupe → areia → taupe (simétrico 90deg) */
  --gradient-text-1:    linear-gradient(90deg, #997E69, #E2AA80, #997E69);
  /* gradient-text2: terracota → salmão → terracota (simétrico 90deg) */
  --gradient-text-2:    linear-gradient(90deg, #9D2D19, #ED7F6C, #9D2D19);
  /* botão .seu-botao-personalizado + footer */
  --gradient-btn:       linear-gradient(to right, #9D2D19, #ED7F6C, #9D2D19);
  /* seção depoimentos (radial) */
  --gradient-testimonial: radial-gradient(at center center, #ED7F6C 0%, #9D2D19 100%);
  /* cards de feature e box de preço (claro) */
  --gradient-card-light: linear-gradient(90deg, #E9E9E9 0%, #FFFFFF 100%);
  /* glow decorativo */
  --gradient-glow:      radial-gradient(ellipse at center, rgba(157,45,25,.28) 0%, transparent 70%);

  /* ── Tipografia ── */
  --font-display: "quincy-cf", Georgia, serif;   /* TypeKit · fw 400/600 */
  --font-body:    "montserrat", Arial, sans-serif; /* fw 400/500/700 */

  /* ── Escala Tipográfica ── */
  --text-xs:11px; --text-sm:13px; --text-base:15px; --text-md:18px;
  --text-lg:22px; --text-xl:28px; --text-2xl:36px; --text-3xl:48px;
  --text-4xl:60px; --text-hero:76px;

  /* ── Espaçamento (base 4px) ── */
  --space-1:4px; --space-2:8px; --space-3:12px; --space-4:16px;
  --space-5:20px; --space-6:24px; --space-8:32px; --space-10:40px;
  --space-12:48px; --space-16:64px; --space-20:80px;

  /* ── Border Radius ── */
  --radius-sm:4px; --radius-md:8px; --radius-lg:16px;
  --radius-xl:24px; --radius-pill:9999px;

  /* ── Sombras ── */
  --shadow-glow: 0 0 40px rgba(157,45,25,.40);
  --shadow-warm: 0 8px 32px rgba(226,170,128,.30);

  /* ── Layout ── */
  --container-max: 1140px;  /* Máximo Elementor padrão */
  --grid-gap: 20px;         /* --widgets-spacing */
}

/* ── Classes de gradient text (usadas no site) ── */
.gradient-text1, .gradient-text2 {
  -webkit-background-clip: text; background-clip: text; -webkit-text-fill-color: transparent;
}
